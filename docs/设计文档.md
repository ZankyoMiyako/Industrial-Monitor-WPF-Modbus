# 工业监控系统设计文档

## 项目目标
开发一个基于WPF和Modbus协议的工业设备监控系统，用于学习和展示工业软件开发能力。

## 技术选型
### 前端技术
- WPF (Windows Presentation Foundation)
- Prism框架 (MVVM模式)
- Material Design UI库

### 通信协议
- Modbus TCP
- 使用NModbus库

### 开发环境
- Visual Studio 2022
- .NET 8.0
- Git + GitHub

## 功能模块
1. 设备连接管理
2. 实时数据监控
3. 历史数据查询
4. 报警管理
5. 用户界面
